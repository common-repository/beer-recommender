== Description ==

With this Wordpress Plugin you can easily discover new beers!
Just add it to your pages with the shortcode or use it as a widget.

For the API Access you have to register at brewerydb.com and insert your API-Key in the plugin.


== Installation ==



1. Upload `beerrecommender` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the shortcode via the editor in your pages (Beer Icon)
